# What is a Region 
Region - Physical location where cloud resources are hosted.

# What is an Availability Zone
An Availability Zone is a separate data center inside a cloud region used for high availabity and fault tolerance.

# Why AWS uses multiple AZs
AWS uses multiple AZs so systems stay online, reliable, and protected from failure(High Available,Fault tolerance,Disaster recovery).

# Basic idea of global infrastructure
Global infrastructure is the worldwide setup of data centers and networks that deliver cloud services globally.
